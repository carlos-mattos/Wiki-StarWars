## 🔖 Sobre
<b>Wiki-StarWars</b> é uma aplicação web desenvolvida em React, voltada para os(as) nerds fãs da saga. Com uso de uma [API](https://swapi.dev/) externa, fizemos uma pequena enciclopédia do universo Star Wars.

## 🗂 Como baixar e iniciar o projeto

```bash
# Abra um terminal e copie este repositório com o comando
$ git clone https://github.com/carlos-mattos/Wiki-StarWars.git
# ou use a opção de download.

# Entre na pasta Wiki-StarWars
$ cd Wiki-StarWars

# Instale as dependências
$ npm install

# Inicie o React
$ npm run start

```
